More and more code samples are stored on AuthWiki but for non-developers it can be confusing to understand how to actually get content and download files from Code wIki. Yes, each file could be viewed and its content copy-pasted to a local file, but that may not be the best solution.

To download from AuthWiki, scroll to the bottom left corner of the code sample there download button will be visible on the right. Click on the button and a ZIP file will containing the entire code sample would be downloaded on your device.
